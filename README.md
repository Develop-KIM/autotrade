# AutoTrading Bot

## 소개
이 프로젝트는 Python을 사용하여 AI 기반 비트코인 자동매매 시스템을 구현하는 프로젝트입니다. <br/>
MVP 단계로 시작하여 점진적으로 기능을 확장합니다.

## 설치 및 실행

### 1. 가상환경 설정
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 실행
```bash
python src/autotrade.py
```
